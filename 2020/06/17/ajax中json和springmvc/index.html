ajax中json和SpringMVC

发表于 2020-06-17 | 分类于 框架

json

json对象格式:{"param":"value","param":"value"}

普通js对象:{param:"value",param:"value"}

js对象转json对象:

  • 先将js转json字符串：JSON.stringify()
  • 将json字符串转json对象:JSON.parse()

ajax中的contentType

ajax中，没有指定contentType的话，Jquery默认使用application/x-www-form-urlencoded类型，data使用json对象格式,springmvc可以@RequestParam接收数据(变量同名不需要)

指定contentType为application/json时，data必须为json字符串格式，springmvc接收数据要用@RequestBody注解

$.ajax({
    type: "post",
    url:  "mobile/notice/addMessageInfo.jspx",
    contentType: "application/json",
    data:"{'name':'zhangsan','age':'15'}",
    dataType: "json",
    success: function(data) {
        console.log(data);
    },
    error: function(msg) {
        console.log(msg)
    }
})
